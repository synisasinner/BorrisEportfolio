Implementing NLP Application Models using spaCy

Overall Grade

95
Aim High!

Learning Reflection

This activity on the use of spaCy to implement NLP models was worth knowledge for learning. It used applications that need careful data preparation and tuning, like POS tagging, NER, and text classification. Pre-trained pipelines of SpaCy really facilitated simple and efficient model integration like en_core_web_sm and en_core_web_lg. I experimented with different datasets, and thereby saw how preprocessing and quality of data affect the model's accuracy. Overall, the project demonstrated how critical it is to align tasks, data, and metrics for significant results.
